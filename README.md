# Machine Learning Time Series

Este repositório reúne meus estudos sobre **Análise e Previsão de Séries Temporais (Time Series Analysis)** utilizando Python. O projeto explora os principais conceitos estatísticos e computacionais empregados na análise de dados temporais, desde a exploração inicial da série até a construção de modelos clássicos de previsão.

O objetivo é compreender como identificar padrões, tendências, sazonalidade e dependências temporais, aplicando técnicas amplamente utilizadas em Machine Learning, Estatística e Ciência de Dados.

---

## 📖 Sobre o projeto

Séries temporais estão presentes em praticamente todas as áreas da engenharia e da ciência de dados, como previsão de demanda, monitoramento industrial, mercado financeiro, meteorologia, saúde e análise de sensores.

Neste projeto são desenvolvidos estudos práticos utilizando Python para compreender o comportamento de dados ao longo do tempo, explorando conceitos fundamentais para preparação, análise e previsão de séries temporais.

## ▶️ Como executar

1. Clone este repositório.

```bash

```

2. Instale as dependências.

```bash
pip install -r requirements.txt
```

3. Execute os notebooks utilizando o Jupyter Notebook ou JupyterLab.
